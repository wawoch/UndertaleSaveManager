    - Изменить кол-во золота
    
    - Изменить LV
    
    - Изменить HP
    
    - Изменить имя персонажа
    
    - Изменить кол-во опыта
    
    - Изменить кол-во убийств
    
    - Выдать зонт
    
    - Разрушить барьер
    
    - Геноцид
    
    - Изменить ID текущей комнаты
